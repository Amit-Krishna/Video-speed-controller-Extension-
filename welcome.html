Welcome to Video Speed Controller!

You're all set up to control video speed on any website. Here are the key features to get you started:

Key Features

  • On-Page Controller: A controller will appear when a video is present. Just hover over the video to see it! It will hide automatically when you're not using it.
  • Keyboard Shortcuts: Control speed without ever leaving the video player.
  • Per-Site Defaults: Set your preferred speed for different websites in the Options page.

Default Shortcuts

You can change these anytime from the Chrome Extensions shortcut page.

  • Increase Speed: Ctrl+Shift+.
  • Decrease Speed: Ctrl+Shift+,
  • Reset Speed: Ctrl+Shift+0
Go to Options Configure Shortcuts
